# Handmade

Handmade by Roshani is a digital space showcasing exquisite, handcrafted crochet pieces. From cozy clothing and elegant home decor to unique accessories, every creation is crafted with passion, dedication, and premium quality materials.

## ✨ Features

- **Artisanal Showcase**: A curated gallery featuring featured collections of handmade crochet items.
- **Custom Orders**: A dedicated contact system for users to request personalized artisan pieces.
- **Premium Aesthetics**: A modern, responsive design built with a focus on visual excellence and user experience.
- **Dynamic Content**: Category-based exploration including Clothing, Home Decor, and Accessories.

## 🛠️ Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Components**: [Shadcn UI](https://ui.shadcn.com/)
- **Runtime**: [Bun](https://bun.sh/)
- **Icons**: [Lucide React](https://lucide.dev/)


### Prerequisites

- [Bun](https://bun.sh/) runtime installed on your machine.

### Installation

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd handmade
   ```

2. **Install dependencies**:
   ```bash
   bun install
   ```

3. **Run the development server**:
   ```bash
   bun run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📂 Project Structure

- `app/`: Next.js App Router pages and global styles.
- `components/`: Reusable UI components (Shared, Layout, and Pages).
- `hooks/`: Custom React hooks.
- `lib/`: Utility functions and shared library configurations.
- `public/`: Static assets including logos and icons.
